# Library Management System

A Node.js backend for a Library Management System. This project provides RESTful APIs for managing books, users, and issued books, with role-based access for admins and librarians.

## Features
- User authentication and authorization
- Admin and librarian roles
- Book management (add, update, delete, list)
- Issue and return books
- Email notifications
- Error handling and utilities


## API Endpoints
- Admin routes: `/admin`
- Librarian routes: `/librarian`
- Book routes: `/books`
